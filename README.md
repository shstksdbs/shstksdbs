## 💡 How to Work
- 단순 구현이 아닌 **원리와 구조까지 이해**하려는 태도로 안정적이고 확장성 있는 코드를 지향합니다.
- 프로젝트 과정에서 발생하는 문제는 **끝까지 책임지고 해결**하는 자세로 임합니다.
- 협업 시 **명확한 소통과 역할 분담**을 중시하여 팀 전체의 효율성을 높입니다.
- 업무 진행 중 발견한 오류나 개선점은 **즉시 피드백하고 보완**하여 품질을 유지합니다.
